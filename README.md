# Topomapper

Topomapper erstellt aus Höhenlinien- und Topodaten stilisierte Kartenmotive und bereitet sie für den Druck oder die Weiterverarbeitung auf.

## Funktionen

- Import von Topodaten und Höhenlinien
- Generierung stilisierter Karten- und Liniengrafiken
- Export als druckfähige Dateien
- Konfigurierbare Farbpaletten und Layouts

## Voraussetzungen

- Node.js (empfohlen: aktuelle LTS-Version)
- npm oder ein kompatibler Paketmanager

## Installation

```bash
npm install
```

## Nutzung

```bash
npm run start
```

Die Anwendung startet anschließend lokal. Weitere Konfigurationsoptionen sind in den Quelltexten dokumentiert.

## Projektstruktur

- `src/` – Anwendungslogik und Verarbeitung der Topodaten
- `public/` – Statische Assets
- `scripts/` – Hilfsskripte für Verarbeitung und Export

## Entwicklung

```bash
npm run dev
```
